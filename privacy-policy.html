Privacy Policy

Extension: Roblox Mutual Friends Checker
Effective date: April 12, 2026

This privacy policy describes how the browser extension Roblox Mutual Friends Checker (“the Extension”) handles information when you use it. The Extension is offered as a convenience tool and is not affiliated with Roblox Corporation.

Summary

The Extension does not send your data to servers operated by the Extension’s developer. It runs on your device and communicates only with Roblox’s official API domains (users.roblox.com and friends.roblox.com) to perform username lookups, load friend lists, and display mutual friends in the extension popup.

Information you provide

When you use the Extension, you may enter two Roblox usernames. Those usernames are used only to request public or account-appropriate data from Roblox’s APIs as needed to show results. They are not uploaded to a separate database or analytics service by the Extension.

How data is used

  • Username resolution: The Extension sends the usernames you enter to Roblox’s user API to obtain user identifiers and profile fields needed to display names and links.
  • Friend lists: The Extension requests each user’s friend list from Roblox’s friends API so it can determine whether the two users are friends and compute mutual friends.
  • Display: Results are shown only inside the Extension’s popup interface in your browser.

Cookies and Roblox session

Some requests to Roblox’s friends API may include your browser’s existing cookies for Roblox domains (the Extension uses standard browser requests with credentials where the API requires an authenticated session). Those cookies are sent only to Roblox, not to the Extension developer. Whether Roblox treats a request as signed-in depends on your Roblox account and browser state. The Extension does not read passwords or payment information.

Permissions and host access

The Extension is limited to network access to https://users.roblox.com/* and https://friends.roblox.com/* for the purposes above. It does not inject scripts into arbitrary websites or request broad access to all Roblox web pages.

Data storage and retention

The Extension does not use its own remote servers to store what you type. The Extension does not use extension-managed cloud sync for your inputs. Temporary data may exist in your browser’s memory only while you use the popup and is cleared when you close the popup or refresh, consistent with normal browser behavior.

Third parties

Roblox processes requests according to Roblox’s privacy policy. The Extension developer does not sell personal information and does not use advertising or analytics SDKs inside this Extension.

Children’s privacy

Roblox is used by people of many ages. This Extension does not knowingly collect personal information for the developer’s own use. If you are a parent or guardian and have questions about how Roblox handles account data, please refer to Roblox’s policies and parental controls.

Changes

This policy may be updated if the Extension’s behavior changes (for example, new permissions or features). The “Effective date” at the top will be revised when updates are published.
